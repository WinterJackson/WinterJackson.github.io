## üì¶ Installation & Setup

Follow these steps to run the project locally:

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/WinterJackson/WinterJackson.github.io.git
    cd WinterJackson.github.io
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Run Development Server**
    ```bash
    npm run dev
    ```
    Open `http://localhost:5173` (or the port shown in terminal) to view the site.

4.  **Build for Production**
    ```bash
    npm run build
    ```
    The optimized files will be generated in the `dist/` directory.
